    Core Concepts #

    • Prior $P(\theta)$ and likelihood $P(D|\theta)$ form posterior via Bayes Rule.
    • Model uncertainty is critical in embedded ML and safety-critical systems.

    $$ P(\theta|D) = \frac{P(D|\theta)P(\theta)}{P(D)} $$

    Applications #

    • Neural network calibration
    • Monte Carlo simulation
    • Probabilistic inference in robotics
